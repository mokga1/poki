# 포키 (Poki) - 3차선 러너 게임 디자인

작성일: 2026-06-09

## 개요

웹 브라우저에서 플레이할 수 있는 3차선 무한 러너 게임. Subway Surfers 스타일의 뒤에서 보는 시점. 캐릭터는 자동으로 앞으로 달리고, 플레이어는 좌우 차선 이동, 점프, 슬라이드로 장애물을 피하고 코인을 모은다.

## 기술 스택 및 파일 구조

- **렌더링**: Three.js (CDN 로딩, 빌드 도구 불필요)
- **실행 환경**: 최신 데스크톱 브라우저 (Chrome, Edge, Firefox)
- **언어**: HTML + JavaScript (ES modules)

```
index.html       게임 진입점 (Three.js CDN, 캔버스, HUD)
src/game.js      게임 메인 루프, 상태 관리
src/player.js    캐릭터 (외형, 점프/슬라이드/차선 이동)
src/world.js     지면, 장애물/코인 스폰 및 관리
src/input.js     키보드 입력 처리
```

## 화면 / 카메라

- 카메라: 플레이어 뒤쪽 약간 위에서 따라다님 (3인칭)
- 3차선 바닥이 화면 안쪽으로 뻗어 있음
- 좌측 상단 HUD: 현재 점수 (큰 숫자)
- 게임 오버 시 화면 중앙 오버레이: "GAME OVER", 최종 점수, 최고 점수, "Press R to Restart"

## 캐릭터 (여자 캐릭터, A안 - 단순 도형)

기본 3D 도형을 조합해 사람 모양으로 만든다.

| 부위 | 모양 | 색 |
|------|------|-----|
| 머리 | 구 (sphere) | 살구색 |
| 포니테일 | 작은 박스 (머리 뒤에 부착) | 분홍색 |
| 몸통 | 박스 | 분홍색 |
| 팔 (x2) | 얇은 박스 | 살구색 |
| 다리 (x2) | 얇은 박스 | 보라색 |

- 캐릭터는 화면 중앙 근처에 시각적으로 고정
- 달리기 애니메이션: 팔과 다리를 일정 주기로 좌우(앞뒤) 흔드는 간단한 처리
- 차선 이동 시 약 0.2초에 걸쳐 부드럽게 좌우 이동

## 차선 / 장애물 / 코인

### 차선
- 3개 고정 차선 (왼쪽 / 가운데 / 오른쪽), 차선 간격 약 2m
- 플레이어는 항상 3개 차선 중 하나에 위치

### 장애물

| 종류 | 외형 | 행동 | 통과 방법 |
|------|------|------|-----------|
| 정지 기차 | 길쭉한 회색 박스 | 가만히 있음 | 옆 차선으로 피하기 또는 뒤쪽 노란 계단 박스로 올라가기 |
| 움직이는 기차 | 길쭉한 빨간 박스 | 플레이어 쪽으로 천천히 다가옴 | 옆 차선으로 피해야 함 (점프해도 박치기 = 게임오버) |
| 낮은 장애물 (바리케이드) | 짧은 박스 | 정지 | 아래 화살표 길게 눌러 슬라이드 |
| 높은 장애물 (간판) | 위에 떠 있는 박스 | 정지 | 위 화살표로 점프 |

### 기차 위 통과
- 정지 기차 뒤쪽에 노란 계단 박스
- 계단 위에서 점프하면 기차 위 높이로 올라감
- 기차 위에서도 코인 획득 가능
- 기차 끝에 다다르면 자동으로 내려옴

### 코인
- 노란색 작은 구체
- 차선마다 일정 간격, 가끔 5~10개씩 줄지어 배치
- 기차 위에도 가끔 배치
- 닿으면 사라지고 점수 +10

## 조작

| 키 | 동작 | 비고 |
|----|------|------|
| ← | 한 차선 왼쪽으로 이동 | 끝 차선이면 무시 |
| → | 한 차선 오른쪽으로 이동 | 끝 차선이면 무시 |
| ↑ | 점프 (~0.6초) | 공중에서 다시 누르면 무시 |
| ↓ (길게 누름) | 슬라이드 (높이 절반) | 떼면 일어남 |
| R | (게임 오버 시) 재시작 | |

- 캐릭터는 키 입력 없이도 항상 앞으로 이동

## 충돌 / 게임 오버

즉시 게임 오버:
- 정지 기차 정면에 부딪힘
- 움직이는 기차에 부딪힘 (점프해서 박치기해도 게임오버)
- 낮은 장애물에 슬라이드 없이 부딪힘
- 높은 장애물에 점프 없이 부딪힘

게임 오버 시 화면 어두워지고 결과 표시. R 키로 즉시 재시작 (페이지 리로드 없이).

## 점수 및 진행

### 점수
- 거리 점수: 1초마다 +1점
- 코인 점수: 코인 1개당 +10점
- 화면 좌측 상단에 실시간 표시

### 난이도
- 시작 속도: 반응 가능한 보통 속도
- 10초마다 속도 약간 증가, 최대 2배까지
- 속도가 올라갈수록 장애물 간격도 약간 좁아짐

### 최고 점수
- 브라우저 `localStorage`에 저장
- 게임 오버 화면에 "최고 점수: XXXX" 표시
- 갱신 시 "신기록!" 표시

## 모듈 분리 원칙

각 모듈은 한 가지 역할만 가진다:

- `input.js`: 키 상태만 노출 (`isLeftPressed()`, `isJumpPressed()` 등). 다른 게임 로직 모름.
- `player.js`: 플레이어 상태(위치/차선/점프/슬라이드)와 외형. 입력은 game.js가 주입.
- `world.js`: 지면, 장애물, 코인 스폰/이동/제거. 충돌 판정 결과만 반환.
- `game.js`: 위 모듈을 묶어 메인 루프 실행. 점수, 게임 오버, 재시작 관리.
