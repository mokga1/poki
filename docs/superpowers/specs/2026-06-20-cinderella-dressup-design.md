# 신데렐라 꾸미기 게임 — 설계 문서

작성일: 2026-06-20

## 개요

기존 러너 게임(포키)과 별개의, 마우스로 즐기는 **신데렐라 꾸미기(dress-up) 게임**.
처음엔 누더기·헝클어진 모습의 신데렐라를 사용자가 드레스·악세서리·화장으로 예쁘게 꾸미고,
마지막에 왕자를 누르면 무도회 엔딩(춤·음악·축하)이 재생된다.

대상: 제작자의 딸. 한국어 UI.

## 목표 / 비목표

**목표**
- 정렬된 종이인형 에셋을 겹쳐 쌓아 꾸미기를 구현
- 옷장 / 서랍장 / 페이스페인팅 / 왕자 4개 버튼 흐름
- 무도회 엔딩(연출 + 음악)
- 기존 러너 게임과 공존하는 폴더 구조(게임 선택 허브)

**비목표 (YAGNI)**
- 저장(localStorage), 다국어, 모바일 터치 최적화는 이번 범위 밖
- 왕자 캐릭터 갈아입히기(엔딩 전용 고정 그래픽으로 충분)

## 폴더 구조 (게임 허브)

```
index.html              ← 게임 선택 메인 (포키 러너 / 신데렐라)
games/
  runner/               ← 기존 포키 게임 이동 (index.html + src/)
  cinderella/
    index.html
    assets/             ← 다운로드한 에셋 (이미 존재)
    src/
      data.js           ← 카테고리/슬롯별 아이템 목록·레이어 순서 정의
      doll.js           ← 종이인형 레이어 렌더링 및 슬롯 교체
      ui.js             ← 좌측 버튼, 우측 패널, 하위 탭
      ending.js         ← 무도회 엔딩 연출
      sound.js          ← 웹오디오 음악(배경/왈츠)
      main.js           ← 초기화·상태 연결
```

- 기존 러너 게임을 `games/runner/`로 이동(루트 `index.html`/`src/` → `games/runner/`로).
- 루트 `index.html`은 두 게임을 고르는 단순 메뉴로 새로 작성. 이동 후 러너가 그대로 동작하는지 확인.

## 에셋 / 레이어 방식

- 출처: itch.io MaelleMarylloup "DOLL 컬렉션" 4개 팩.
  - `Doll DressUp Assets` (기본 몸, CC0)
  - `Doll outfits(feminine casual)` (드레스·머리·쥬얼리 등)
  - `Outfits` (Fantasy 250+, 드레스·악세서리·배경 등)
  - `Face elements` (눈·눈썹·속눈썹·입·코·얼굴장식)
- 라이선스: 기본 몸 CC0, 나머지 "개인·상업 자유 사용, 에셋 단독 재판매 금지". 개인 게임 사용에 적합.
- 모든 이미지가 동일한 800×1200 캔버스에 위치 정렬됨 → **별도 좌표 계산 없이 전체 캔버스로 겹쳐 쌓기**만으로 정렬됨.
- 포맷 혼재: PNG, WebP 모두 `<img>`로 처리 가능.

**레이어 순서 (아래 → 위, z-index)**
```
뒷머리 → 몸(base) → 속옷 → 드레스 → 신발
→ 얼굴(눈 → 눈썹 → 속눈썹 → 입 → 볼터치)
→ 앞머리 → 목걸이/몸장식 → 귀걸이 → 티아라/면사포 → (손 아이템)
```
- 각 "슬롯"은 동시에 1개 아이템만 표시. 같은 슬롯의 다른 아이템 선택 시 교체.
- 일부 슬롯(예: 면사포)은 "없음" 선택 가능.

## 화면 구성 (레이아웃 B)

- 가운데~왼쪽: 신데렐라 전신(800×1200 비율을 화면 높이에 맞춰 축소)
- 왼쪽 세로 버튼 4개: 👗 옷장 / 💍 서랍장 / 💄 페이스페인팅 / 🤴 왕자
- 오른쪽 패널: 선택한 버튼의 아이템 그리드(썸네일). 상단에 하위 탭.
- 아이템 클릭 → 즉시 인형에 반영.

### 버튼별 구성

| 버튼 | 하위 탭 | 슬롯/아이템 |
|---|---|---|
| 👗 옷장 | 드레스 / 머리 / 면사포 | 드레스(약 15), 머리(앞+뒤 세트 10), 면사포·왕관 |
| 💍 서랍장 | 목걸이 / 귀걸이 / 반지·팔찌 | 쥬얼리 에셋(목걸이·귀걸이·반지·팔찌 등) |
| 💄 페이스페인팅 | 눈 / 입술 / 볼터치 / 눈썹 | 눈4, 입14, 눈썹10, 속눈썹7, 얼굴장식 |
| 🤴 왕자 | — | 클릭 시 엔딩 시작 |

- 면사포/왕관 에셋이 없으면 간단한 SVG 오버레이로 대체.

## 시작(누더기) → 꾸민 모습

- 초기 상태: 칙칙한 기본 옷 + 기본(헝클어진 느낌) 머리 + 시무룩한 입 + 화장/악세서리 없음.
- 전체 인형 위에 옅은 흐림/먼지 오버레이(반투명)로 "엉망" 느낌 부여.
- 사용자가 항목을 바꿀수록 화사해지고, 핵심 항목(드레스·머리·얼굴)을 갖추면 흐림 오버레이가 사라지는 연출.
- **항상 옷을 입힌 상태 유지**: 시작부터 옷/속옷을 입혀 맨몸이 노출되지 않게 한다(기본 몸은 성인 마네킹이므로).

## 왕자 클릭 → 무도회 엔딩

- 화면이 무도회장으로 전환(페이드).
- 배경: 샹들리에·홀(에셋 배경 또는 SVG). 하객 실루엣이 박수, 공주 실루엣 몇 명 등장.
- 가운데: 사용자가 꾸민 신데렐라 + 왕자(간단 SVG)가 마주보고 살랑살랑 춤(좌우 스웨이 애니메이션).
- 꽃잎/반짝이 입자 낙하, "축하해요! 🎉" 문구.
- 왈츠 음악 재생. "다시 꾸미기" 버튼으로 복귀.

## 음악

- 웹오디오(AudioContext)로 멜로디 생성 — 외부 파일/라이선스 불필요(러너의 sound.js 접근 방식 참고).
- 꾸미는 중: 잔잔한 배경음(선택). 엔딩: 왈츠.
- 🔊 음소거 토글 제공.

## 기술 스택 / 제약

- 순수 HTML/CSS/JS. 빌드 도구 없음. 러너의 Three.js는 사용하지 않음(2D 이미지 레이어).
- 마우스 클릭 조작, PC 웹 기준.
- 모듈은 위 폴더 구조처럼 책임별로 작은 파일로 분리.

## 구현 시 확인 필요 (열린 항목)

- 면사포/왕관/무도회 배경에 해당하는 구체 파일 번호는 구현 단계에서 이미지를 펼쳐 확인 후 선택. 없으면 SVG로 그림.
- "칙칙한 시작 옷"으로 쓸 드레스 번호도 이미지 확인 후 선택.
